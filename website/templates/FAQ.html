Что такое enPlans?

Ключевая особенность: enPlans автоматизирует 95% рутинных расчетов, позволяя сосредоточиться на стратегических решениях.

Как начать работу с системой?

Начать работу с enPlans просто даже для новичков. Следуйте этому пошаговому руководству, и вы сможете создать свой первый план за 15 минут.

1

Регистрация и настройка аккаунта

Используйте электронную почту для регистрации. После подтверждения укажите личную информацию.

2

Выбор организации

Выберите организацию за которую вы будете отчитываться, это может быть: предприятие, министерство или регион.

3

Создание первого плана

Перейдите на страницу "Создать план", укажите год и заполните необходимые данные.

🎯

Совет для начала: Начните с просмотра демо, чтобы ознакомиться с интерфейсом и основными функциями без риска.

Создание плана энергоэффективности

Создание плана энергоэффективности в enPlans — это простой пошаговый процесс, который занимает всего несколько минут. Система проведет вас через все необходимые этапы.

1

Переход к созданию плана

В главном меню выберите "Создать" или нажмите соответствующую кнопку на панели управления на странице Планы.

2

Выбор типа плана

Определите категорию организации по объему энергопотребления:

📋
До 25 тыс. т. у. т.

Упрощенный формат отчетности для организаций с меньшим потреблением

📊
Более 25 тыс. т. у. т.

Расширенный формат с детальными расчетами для крупных потребителей

Выбор влияет на: структуру экспорта, требования к данным, детализацию отчетности и процесс аудита.

3

Выбор базовых параметров

Укажите обязательные параметры: год, показатель энергосбережения, задания по экономии ТЭР, долю местных ТЭР и ВИЭ.

4

Заполнение данных

Внесите информацию о энергопотреблении, заполнив таблицы направлений, мероприятий и показателей.

Аудит планов энергоэффективности

Аудит планов в enPlans выполняется специально назначенными аудиторами — региональными управлениями по надзору за энергоэффективностью.

Роль аудиторов в системе

Кто осуществляет проверку?

Официальные представители региональных управлений по надзору за энергоэффективностью Республики Беларусь

Их полномочия и доступы

Полный доступ ко всем таблицам, расчетам и исходным данным плана для проведения официальной проверки
1

Доступ к плану на проверку

Аудитор получает план в статусе "Не просмотренные" в своем интерфейсе.

2

Проверка данных

Аудитор последовательно проверяет основные показатели, таблицы данных, мероприятия по экономии и местным ТЭР.

3

Вынесение вердикта

По результатам проверки аудитор выбирает один из вариантов:

Есть ошибки
Указываются конкретные ошибки, план возвращается на доработку
Одобрить
План соответствует требованиям, переводится в статус "Одобрен"
Отменить
План переходит в изначальное состояние для аудитора "Не просмотренный"

Посмотрите как проверять планы

Что происходит после проверки?

✗

Если есть ошибки

План возвращается респонденту с комментариями. Статус меняется на "Требует доработки"

✓

Если одобрено

План переводится в статус "Одобрен" и считается официально принятым

↩️

Если отменено

Статус плана возвращается к изначальному для аудитора

Экспорт плана

enPlans предоставляет несколько форматов экспорта для различных целей: от аналитики в Excel до официальных проектов плана для подачи.

1

Переход к экспорту

В меню плана выберите "Экспорт" или перейдите в раздел "Экспорт данных" в главном меню.

2

Настройка параметров экспорта

Выберите формат файла (.xlsx, .pdf, .xml), используйте фильтры по статусу и году, отметьте нужные планы.

3

Формирование и скачивание

Нажмите кнопку "Экспортировать". Система автоматически сформирует файл и предложит его скачать.

📈

Проект плана (.xlsx)

Для анализа и работы с данными. Полные таблицы планов и мероприятий.

📄

Проект плана (.pdf)

Для официальной подачи. Форматированный проект плана, готов к печати.

⚙️

Проект плана (.xml)

Для интеграции с программами. Структурированные данные, машинно-читаемый формат.

Как пройти проверку сертификата ЭЦП

Для отправки планов в системе enPlans требуется использование электронной цифровой подписи (ЭЦП).

💡

Важное требование: Для осуществления функции "отправить план" должно быть установлено ПО "Менеджер сертификатов" от Авест с настроенным сертификатом ЭЦП.

✅

Действительный сертификат

Срок действия сертификата является удовлетворительным

❌

НЕдействительный сертификат

Срок действия сертификата истек, требуется обновление

1

Откройте ПО "Менеджер сертификатов"

Запустите программу "Менеджер сертификатов" от Авест.

2

Выберите действительный сертификат

Выберите "Действительный" сертификат из списка.

3

Экспортируйте сертификат в файл

Кликните правой кнопкой по сертификату и выберите "Экспорт сертификата в файл".

4

Импортируйте сертификат в систему

При отправке плана используйте полученный файл сертификата.
