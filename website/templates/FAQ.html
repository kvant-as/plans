Создание плана энергоэффективности

Создание плана энергоэффективности в EnergoPlans — это простой пошаговый процесс, который занимает всего несколько минут. Система проведет вас через все необходимые этапы.

Переход к созданию плана

В главном меню выберите "Создать" или нажмите соответствующую кнопку на панели управления на странице Планы.

Выбор типа плана

Определите категорию организации по объему энергопотребления:

📋
До 25 тыс. т. у. т.

Упрощенный формат отчетности для организаций с меньшим потреблением

📊
Более 25 тыс. т. у. т.

Расширенный формат с детальными расчетами для крупных потребителей

Выбор влияет на: структуру экспорта, требования к данным, детализацию отчетности и процесс аудита.

Выбор базовых параметров

Укажите обязательные параметры:

  • Год — выберите год из доступных вариантов (система покажет предупреждение, если план на этот год уже существует)
  • Показатель энергосбережения (%) — планируемый процент экономии энергии
  • Задания по экономии ТЭР (т. у. т.) — целевой показатель экономии в тоннах условного топлива
  • Доля местных ТЭР в КПТ (%) — процент использования местных топливно-энергетических ресурсов
  • Доля ВИЭ в КПТ (%) — процент использования возобновляемых источников энергии

Аудит планов энергоэффективности

Аудит планов в EnergoPlans выполняется специально назначенными аудиторами — региональными управлениями по надзору за энергоэффективностью. Аудиторы проводят проверку всех таблиц и данных плана, после чего выносят официальный вердикт.

👔

Роль аудиторов в системе

Кто осуществляет проверку?

Официальные представители региональных управлений по надзору за энергоэффективностью Республики Беларусь

Их полномочия и доступы

Полный доступ ко всем таблицам, расчетам и исходным данным плана для проведения официальной проверки и вынесения заключения

Доступ к плану на проверку

Аудитор получает план в статусе "Не просмотренные" в своем интерфейсе. Доступны все разделы плана для детального изучения.

Проверка данных

Аудитор последовательно проверяет:

  • Основные показатели — корректность расчетов энергосбережения и экономии
  • Таблицы данных — полнота и правильность заполнения всех форм и разделов
  • Данные респондента — точность информации об организации и ответственных лицах
  • Мероприятия по экономии — обоснованность и реалистичность предложенных мер
  • Мероприятия по местным ТЭР — корректность планов по увеличению использования местных ресурсов

Вынесение вердикта

По результатам проверки аудитор выбирает один из вариантов:

Есть ошибки

Указываются конкретные ошибки, план возвращается на доработку

Одобрить

План соответствует требованиям, переводится в статус "Одобрен"

Отменить

План переходит в изначальное состояние для аудитора "Не просмотренный"

Экспорт данных и отчетов

EnergoPlans предоставляет несколько форматов экспорта для различных целей: от аналитики в Excel до официальных отчетов для подачи. Все форматы создаются автоматически на основе ваших данных.

Переход к экспорту

В меню плана выберите "Экспорт" или перейдите в раздел "Экспорт данных" в главном меню.

Настройка параметров экспорта

  • Формат файла — выберите из трех вариантов:
    • Excel (.xlsx) — табличные данные для анализа
    • Отчет (.pdf) — оформленный отчет для печати и подачи
    • Данные (.xml) — структурированные данные для программ
  • Фильтрация планов — используйте фильтры "Статус" и "Год" для выбора
  • Выбор планов — отметьте нужные планы или выберите "Выбрать все"

Формирование и скачивание

После выбора параметров нажмите кнопку "Экспортировать". Система автоматически сформирует файл в выбранном формате и предложит его скачать.

📊

Структура экспорта данных:

2
Часть 2 — Мероприятия по экономии энергоресурсов

Включается при наличии заполненных мероприятий по энергосбережению

3
Часть 3 — Мероприятия по увеличению местных ТЭР

Включается при наличии мероприятий по использованию местных топливно-энергетических ресурсов

Примечание: Если разделы не заполнены в плане, они не будут включены в экспортируемый файл. Система формирует только актуальные данные.

Как пройти проверку сертификата ЭЦП

Для отправки отчетов в системе EnergoPlans требуется использование электронной цифровой подписи (ЭЦП). Этот раздел поможет вам правильно настроить и использовать сертификат ЭЦП.

💡

Важное требование: Для осуществления функции "отправить отчет" должно быть установлено ПО "Менеджер сертификатов" от Авест с настроенным сертификатом электронной цифровой подписи (ЭЦП).

✅

Действительный сертификат

Срок действия сертификата на текущий момент является удовлетворительным

❌

НЕдействительный сертификат

Срок действия сертификата уже закончился, требуется обновление

Откройте ПО "Менеджер сертификатов"

Запустите программу "Менеджер сертификатов" от Авест. Вы увидите интерфейс с доступными сертификатами.

ПО Менеджер сертификатов
